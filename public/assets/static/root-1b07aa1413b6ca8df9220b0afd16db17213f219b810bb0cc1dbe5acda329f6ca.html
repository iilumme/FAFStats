What is FAFStats?

FAFStats is a statistics, utilities and advanced stalking assistance tool for the FAForever community.

FAFStats only supports Ladder 1v1 games and we are not planning on expanding.

Known issues

  • * Login doesn't work. In the current workaround you should be able to log into anyones profile for testing purposes
  • * Sometimes going to previous page doesn't load the page content correctly
  • * The page has to be reloaded after logging in
  • * Rating Evolution & Faction Distribution charts are not currently functional
  • * The FAF API doesn't always return CORS header, so some stuff may seem buggy...

Contribution

We are extremely reliant on the official FAF API , the best way to support FAFStats is to contribute to the API.

Planned functionality

  • * A betting system, where users will be able to bet some sort of points on high level ladder matches
